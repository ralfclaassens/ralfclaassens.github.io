## What is Rails? Rails is a web application framework designed to work with the Ruby programming language.
## Why Rails? * Built-in ORM (ActiveRecord) * All-in-one * Convention over Configuration * Don't Repeat Yourself (minimal code) * Rails conventions lead to less code and less repetition * Mature * Baked in testing
